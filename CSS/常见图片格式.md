# 常见图片格式
- 三原色:红(Red)、绿(green)、蓝(blue),颜色R(red)G(green)B(blue)就是由三原色组成
- gif
    - 支持动画
    - 只有全透明和不透明两种模式
    - 只有256种颜色
- jpg
    - 采用有损压缩算法
    - 体积较小
    - 不支持透明
    - 不支持动画
- png 
    - 采用无损压缩算法
    - 体积相对较小
    - 支持背景透明
    - 不支持动画
- svg
    - 不会失帧
    - 保存图片的信息:默认半径、角度、颜色
    - 一般用于保存颜色形状相对简单的图像
