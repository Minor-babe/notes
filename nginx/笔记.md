一、安装nginx

```shell
sudo apt install nginx
```

二、nginx自动安装后配置文件在那？

```shell
/ect/nginx/
```

三、nginx常用命令

```shell
#启动
sudo nginx start
#重新加载配置
sudo nginx reload
#重启
nginx -s reopen
#停止
nginx -s stop
#查看版本号
nginx  -v
# 检查配置文件是否正确
nginx  -t
```
