# 表单

## 属性

| type属性 |       含义       |
| :------: | :--------------: |
|   text   | 普通的文本输入框 |
| password |    密码输入框    |
| checkbox |      复选框      |
|  radio   |      单选框      |
|   file   |     上传文件     |
|  submit  |     提交按钮     |
|  reset   |     重置按钮     |

## 表单标签

|          |     说明     |
| :------: | :----------: |
|  input   |    输入框    |
| textarea |    文本域    |
|  select  |     选项     |
|  label   | 用于辅助表单 |
