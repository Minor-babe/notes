# 表格

- \<table>:表格的最外层容器

- \<tr>:定义表格行

- \<th>:定义表头

- \<td>:定义表格单元

- \<caption>:定义表格标题

- 属性

|   属性名    |     属性作用     |
| :---------: | :--------------: |
|   border    |     表格边框     |
| cellpadding |  单元格内的空间  |
| cellspacing | 单元格之间的空间 |
|   rowspan   |      合并行      |
|   colspan   |      合并列      |
|    align    |   左右对齐方式   |
|   valign    |   上下对齐方式   |
